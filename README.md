# meindeinunser

Nachbarschafts-Sharing-Plattform fuer Mainz. Verleihen, Verschenken, Suchen.

## Tech-Stack

- Next.js 16 (App Router, TypeScript)
- Supabase (Auth, PostgreSQL, Storage)
- Tailwind CSS v4
- Vercel (Hosting)

## Entwicklung starten

```bash
npm install
npm run dev
```

Oeffne [http://localhost:3000](http://localhost:3000) im Browser.

## Umgebungsvariablen

Kopiere `.env.example` nach `.env.local` und trage die Supabase-Zugangsdaten ein.
